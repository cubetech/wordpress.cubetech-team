== Description ==

cubetech Team - a very easy plugin for managing and listing team members in groups and with shorttags

Plugin features:

* Custom post type
* Short code
* Short code button in TinyMCE
* Uses builtin jQuery
* Reuse team members in different groups
* Use groups for adding team members
* Image hover effect
* Template setting
* Facebook and Xing integration (coming soon)

== Installation ==                                                                                                                                                
                                                                                                                                                                  
This section describes how to install the plugin and get it working.                                                                                              
                                                                                                                                                                  
1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress                                                                                                    
1. Place the shortcode in your content
